# madara-playground

Collection de 5 bots Social Media AI propulsés par Claude.

## Bots disponibles

| # | Bot | Description |
|---|-----|-------------|
| 1 | **Architecte de Stratégie Social Media** | Analyse ton business et construit une stratégie complète |
| 2 | **Constructeur de Piliers de Contenu** | Crée 5 piliers de contenu avec 10 idées par pilier |
| 3 | **Calendrier de Contenu 30 Jours** | Génère un planning de contenu sur 30 jours |
| 4 | **Créateur de Posts qui Arrêtent le Scroll** | Crée des posts ultra-engageants avec 3 variations |
| 5 | **Script Vidéo Courte Virale** | Rédige des scripts pour Reels/TikTok/Shorts |

## Installation

```bash
npm install
```

## Configuration

Définis ta clé API Anthropic :

```bash
export ANTHROPIC_API_KEY=ta-cle-api
```

## Utilisation

### Mode interactif (menu de sélection)

```bash
npm start
```

### Lancer un bot spécifique

```bash
npm run bot:strategie
npm run bot:piliers
npm run bot:calendrier
npm run bot:posts
npm run bot:video
```
